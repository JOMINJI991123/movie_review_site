# 영화 리뷰 사이트

## 🖥️ 프로젝트 소개
사용자가 입력한 리뷰데이터를 분석하여 자동으로 별점을 부여하여 리뷰를 등록하는 사이트

## 🕰️ 개발 기간
* 2023.9.1 - 2023.12.13

### 🧑‍🤝‍🧑 맴버구성
 - 손기완(조장) : 모델 구현 및 튜닝, 팀 일정 조율
 - 한진우 : ppt 및 보고서 문서 작성
 - 이태경 : 웹 페이지 디자인 및 구현
 - 조민지 : 서버 구현, 데이터 통신 및 처리
   
### ⚙️ 개발 환경
- `Python 3.11.5`
- `jQuery 3.6.4`
- `tensorflow 2.15.0`
- **IDE** : Visual Studio Code (1.84.2)
- **Framework** : Flask (2.2.2)
